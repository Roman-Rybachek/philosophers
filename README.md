# philosophers

Программа представляет симуляцию гонки за ресурсы между потоками/процессами. Каждая программа принимает четыре аргумента: 
number_of_philosophers, time_to_die, time_to_eattime_to_sleep, number_of_times_each_philosopher_must_eat (опционально). В общем: 
несколько философов сидят за столом, и между каждыми двумя философами лежит по вилки. Философы могут находиться в трех состояниях:
едят, думают, спят. Если философ вовремя не поест, то он умрет. Но чтобы ему поесть, ему нужно иметь две вилки (это условие задачи).
При правильном соотношении аргументов программы, философы должны жить почти неограниченно. <br/><br/>

Тут три программы. В первых двух, философы являются потоками, а в третьей - процессами. В первой, вилками являются мьютексы. Во второй и третьей - семафоры.
